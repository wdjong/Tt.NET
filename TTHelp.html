TypeTest

Do a test

Enter your name so your results can be stored in the History.

Choose and original to type from. (You can print it out if you like).

Choose how long you want the test to run (in seconds) by choosing Preferences from the Copy menu.

Start to type in the Copy window.

Commands

File menu commands

Exit

Closes the program

Original

Choose

Select some text to type from. (You will be marked on how closely yours matches).

View

Arranges the Original and Copy windows to facilitate copying from the version on the screen.

Hide

The Original is hidden by default because originally trainees would type from hard (printed) copies.

Print

Print a copy of the original to type from.

Copy

New

Clear the existing test and start a new one.

Open

Open an existing test file and mark it.

Preferences

Specify the length of tests and whether editing is allowed.

Mark

Find the speed and accuracy of the copy. (You can choose this part way through a test if you get sick of it.)

Save

Save a copy of your typing for later use.

Print

Print a copy of your typing for later use.

Result

Change name

Change the name of the person being tested which is used in adding to the history

History

Test results are stored in the Access database(ttwin.mdb) and can be viewed on the grid.
